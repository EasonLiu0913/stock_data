# 台灣加權指數日 K 線

## 頁面

`public/twse-market-chart.html` 以三個共用日期軸的區塊顯示：

1. 台灣加權指數每日開盤、最高、最低、收盤 K 線。
2. 集中市場每日成交量及成交值。
3. 外資及陸資每日買賣超金額，不含外資自營商。

日 K 主圖同時依每日收盤指數計算並繪製：

- SMA5：黃色
- SMA10：紫色
- SMA20：青色
- SMA60：橘色
- SMA120：粉紅色

每條均線都使用完整歷史資料計算，即使圖表只顯示近 60 或 120 日，區間起點的均線也不會從畫面第一天重新起算。滑鼠 tooltip 會同步顯示當日各條 SMA 數值。

頁面支援：

- 近 60、120、250 日及全部資料。
- 滑鼠滾輪縮放。
- 滑鼠或觸控左右拖曳。
- 同步十字線與每日 tooltip。
- 鍵盤左右鍵移動，`+`、`-` 縮放。
- 桌面及手機版響應式顯示。

## 資料來源

彙整檔：

```text
data_twse_market_chart/market_chart.json
```

官方來源：

| 圖表欄位 | TWSE 報表 | 說明 |
| --- | --- | --- |
| 開高低收 | `MI_5MINS_HIST` | 發行量加權股價指數歷史資料，每次取得一個月 |
| 成交量、成交值 | `FMTQIK` | 市場成交資訊，每次取得一個月 |
| 外資買賣超 | `BFI82U` | 外資及陸資買賣金額，不含外資自營商，每次取得一日 |

外資區塊使用官方買進、賣出「金額」差額，不把不同價格股票的買賣超股數直接相加。

## 產生資料

預設從 2025/11/03 產生到台北當日：

```bash
npm run build:twse-market-chart
```

指定日期：

```bash
node scripts/build_twse_market_chart.js \
  --start-date 20251103 \
  --date 20260724
```

第一次執行會逐日補抓外資買賣超金額。之後腳本會沿用彙整檔中已有的外資資料，只抓尚未存在的交易日；OHLC 與成交量仍會按月重新讀取，以便補上當月最新交易日。

如需強制重新抓取所有外資資料：

```bash
node scripts/build_twse_market_chart.js \
  --start-date 20251103 \
  --date 20260724 \
  --force-foreign
```

## 資料檢查

腳本輸出前會檢查：

- 每一日都有完整 OHLC、成交量、成交值及外資買賣超。
- `MI_5MINS_HIST` 收盤指數與 `FMTQIK` 加權指數相同。
- 最低指數不高於開盤／收盤，最高指數不低於開盤／收盤。
- 所有必要欄位都是有效數字。

任何檢查失敗時不會覆寫正式彙整檔。

## GitHub Actions

`.github/workflows/build-twse-market-chart.yml` 每個台北交易日執行兩次：

- 23:07
- 隔日 02:33

日期採台北時間早上 08:00 切換規則，因此延遲到午夜後執行仍會處理前一交易日。也可以從 GitHub Actions 手動指定 `date` 與 `start_date`。
